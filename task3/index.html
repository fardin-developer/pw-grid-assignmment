The grid-auto-columns Property

Use the grid-auto-columns property to set a default size (width) for all columns.


Grid-auto-rows used to defined the height of row...
Here "grid-auto-rows: 150px 50px;" means height of each first row 150px and second row 50px


Similarly grid-auto-columns: 500px; means width of each columns is 500px
